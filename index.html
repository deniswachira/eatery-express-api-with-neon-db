Overview

Welcome to the EateryExpress API documentation. This API allows you to interact with the EateryExpress platform to manage restaurant data, orders, and more.

Endpoints

GET /restaurants

Retrieve a list of all restaurants.

GET /api/restaurants

GET /restaurants/:id

Retrieve details of a specific restaurant by ID.

GET /api/restaurants/:id

POST /orders

Create a new order.

POST /api/orders

Request Body:

{
    "restaurant_id": "123",
    "items": [
        {"item_id": "abc", "quantity": 2},
        {"item_id": "def", "quantity": 1}
    ],
    "customer_name": "John Doe",
    "address": "123 Main St, Anytown"
}

Examples

Get all restaurants

curl -X GET https://api.eateryexpress.com/api/restaurants

Create an order

curl -X POST https://api.eateryexpress.com/api/orders \
-H "Content-Type: application/json" \
-d '{
    "restaurant_id": "123",
    "items": [
        {"item_id": "abc", "quantity": 2},
        {"item_id": "def", "quantity": 1}
    ],
    "customer_name": "John Doe",
    "address": "123 Main St, Anytown"
}'
