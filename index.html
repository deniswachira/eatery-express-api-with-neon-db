Overview

Welcome to the EateryExpress API documentation. This API allows you to retrieve information about restaurants and cities with restaurants.

Endpoints

GET /restaurants

Retrieve a list of all restaurants.

GET /api/restaurants

GET /restaurants/:id

Retrieve details of a specific restaurant by ID.

GET /api/restaurants/:id

GET /citieswithrestaurants

Retrieve a list of cities that have restaurants.

GET /api/citieswithrestaurants

Examples

Get all restaurants

curl -X GET https://api.eateryexpress.com/api/restaurants

Get a specific restaurant by ID

curl -X GET https://api.eateryexpress.com/api/restaurants/{id}

Get all cities with restaurants

curl -X GET https://api.eateryexpress.com/api/citieswithrestaurants
